# Conversor de Algarismos Romanos (TDD)

[cite_start]Projeto desenvolvido para a disciplina de **Técnicas de Programação 2** na Universidade de Brasília (UnB). [cite_start]O objetivo é converter algarismos romanos para inteiros (até 3000) utilizando a metodologia **Test-Driven Development (TDD)**[cite: 12, 13].

## 🛠️ Tecnologias e Ferramentas
- [cite_start]**Linguagem:** C++14 
- [cite_start]**Framework de Testes:** Google Test (gtest) 
- [cite_start]**Análise de Cobertura:** gcov [cite: 7]
- [cite_start]**Qualidade de Código:** cpplint (Google Style) e cppcheck [cite: 7]
- [cite_start]**Análise Dinâmica:** Valgrind [cite: 7]

## 🚀 Como Executar
[cite_start]O projeto utiliza um `Makefile` para automatizar todas as tarefas.

1. **Compilar e rodar testes:**
   ```bash
   make test
